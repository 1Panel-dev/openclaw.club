OpenClaw 快速安装指南

本文档介绍在 Linux 环境中通过 1Panel 部署 OpenClaw 的标准路径：安装 1Panel、准备模型 API Key、创建智能体并完成飞书渠道对接。

一、环境准备

  • Linux 服务器（Debian / RedHat 系均可）
  • x86_64 或 aarch64 架构
  • 建议可用内存 2GB 以上
  • 可访问互联网
  • 模型 API Key（如 DeepSeek / Kimi / OpenAI）

1Panel 在线安装文档：https://1panel.cn/docs/v2/installation/online_installation/

二、安装 1Panel

步骤 1：获取 root 权限

sudo su -

步骤 2：执行在线安装脚本

bash -c "$(curl -sSL https://resource.fit2cloud.com/1panel/package/v2/quick_start.sh)"

步骤 3：按提示完成初始化

三、准备模型 API Key

以 DeepSeek 为例，访问 https://platform.deepseek.com/ 注册并创建 API Key。

四、部署 OpenClaw

在 1Panel 的 AI > 智能体中，先添加模型账号，再创建 OpenClaw 智能体。

五、飞书渠道配置

进入飞书开放平台创建应用与机器人，配置权限、事件与回调，然后在 1Panel 中填入凭证并保存。
