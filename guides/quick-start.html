OpenClaw Quick Start

This guide walks through the complete path: install 1Panel on Linux, prepare your model API key, deploy OpenClaw, and connect Feishu. If you prefer Chinese, switch to the Chinese version from the top navigation.

Environment Checklist

  • Linux server or client (Debian/RedHat family recommended)
  • x86_64 or aarch64 architecture
  • At least 2 GB available memory
  • Modern browser (Chrome, Firefox, Edge)
  • Internet access
  • One model API key (for example DeepSeek, Kimi, OpenAI, or your local model endpoint)

1Panel installation docs: https://1panel.cn/docs/v2/installation/online_installation/

Install 1Panel

Step 1: Switch to root

sudo su -

Step 2: Run the online installer

bash -c "$(curl -sSL https://resource.fit2cloud.com/1panel/package/v2/quick_start.sh)"

Step 3: Follow interactive prompts

  • Choose install directory (default is usually fine)
  • Install Docker if prompted
  • Configure panel port, security entry, and credentials

Prepare Model API Key

This example uses DeepSeek. You can use any model provider supported by OpenClaw.

Step 1: Register on DeepSeek platform

https://platform.deepseek.com/

Step 2: Create API key

Deploy OpenClaw in 1Panel

Step 1: Add model account

Go to AI > Agent, then add your model provider account first.

Step 2: Create the OpenClaw agent

Set app version, WebUI port, Bridge port, and select your model account.

Step 3: Validate deployment

Open the WebUI from the agent list and verify your assistant can respond normally.

Configure Feishu Channel

Step 1: Create an app in Feishu Open Platform

https://open.feishu.cn/app?lang=en-US

Step 2: Create bot and set permissions

Create your bot, then import the required permission scopes in app settings.

Step 3: Fill credentials in 1Panel

Get app credentials from Feishu and save them in the OpenClaw channel config.

Step 4: Add event subscription and publish

Configure event callback and subscribe to im.message.receive_v1, then publish a new app version.

Switch Models Later

You can switch model settings anytime from AI > Agent > Configuration in 1Panel.
